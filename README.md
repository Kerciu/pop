# pop
Search &amp; optimalization classes @ WUT

## How to run the program?

### 1. Training model

```bash
python3 src/main.py train --episodes <num_of_episodes>
```

### 2. Test the model

```bash
python3 src/main.py eval
```

### 3. Test logic without AI model

```bash
python3 src/main.py eval --autopilot
```
